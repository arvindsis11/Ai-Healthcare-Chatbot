# Product Roadmap

Founder: Arvind Sisodiya

## Phase 1 - AI Healthcare Assistant
- Safe symptom-aware chat.
- RAG with citations.
- Basic triage and specialist recommendation.

## Phase 2 - AI Healthcare API Platform
- Multi-tenant API management.
- OAuth2/JWT security.
- Analytics and billing controls.

## Phase 3 - AI Telemedicine Platform
- Appointment scheduling.
- Doctor-network integration.
- Longitudinal patient conversation context.

## Phase 4 - AI Clinical Decision Support
- Evidence-grounded care pathways.
- Knowledge graph-driven retrieval.
- Auditable AI recommendations for clinicians.
